Pseudo-class e Pseudo-element

Pseudo-class

São usadas para definir propriedades em um estado especifico do elemento.

Por exemplo, pode-se usar este recurso para estilizar um elemento quando se passa o mouse sobre ele, ou quando o usuário clica nele.

Passe o mouse aqui (:hover)

Pseudo-element

Parecido com a pseudo-class, porém o pseudo-element lidará com partes específicas do elemento.

Por exemplo, podmeos usa-lo para estilizar a primeira letra/linha de um elemento, ou para inserir conteúdos antes e depois do elemento.
