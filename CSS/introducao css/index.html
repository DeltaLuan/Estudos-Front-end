Introdução CSS

Com o CSS (Cascading Style Sheets), é possível estilizar um documento HTML de diversas formas.

Podemos mudar a fonte, o fundo, a borda, a cor, entre muitas outras coisas. É muito personalizável!

Podemos até colocar imagens no fundo...

Ao decorrer do tempo, cada item será visto de forma mais particular, então se atente aos novos arquivos que virão no repositório.
