Display

Introdução

Elementos block-level

São os elementos que começam em uma nova linha e pegam toda a largura disponível.

O elemento <div> é um elemento block-level

Elementos inline

São os elementos que não começam uma nova linha e pegam apenas a largura necessária.

Este elemento <span> está embutido dentro de um parágrafo, portanto é um elemento inline

Propriedade display

Estando introduzido no assunto, a propriedade display altera a forma que o elemento é exibido.
Exemplificando, podemos alterar um elemento block-level para que este seja inline, assim como também podemos fazer o inverso

Display: inline

Uma lista inline:

  • Café
  • Chá
  • Suco

Display: block

Este spancom o display: blockfica com uma quebra de linha em casa <span>

Display: none

Este é um h4 visível

Este é um h4 invisível

O h4 com display: none; além de ficar invisível, também não ocupa nenhum espaço
