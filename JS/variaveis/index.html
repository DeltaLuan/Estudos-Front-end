Introdução variáveis

Para declarar uma variável em JavaScript, nós utilizamos o let e/ou também o var - embora que var seja mais "antigo"
Exemplo:
let name = "João";
alert('name'); //irá dar um alerta com o valor que a variável recebeu

Também podemos declarar uma variável como o valor de outra variável:
Exemplo:
let name = "João";
let admin;
admin = name;
alert('admin'); //irá mostrar o valor da varíavel admin, que por sua vez tem como valor a varíavel name

Por fim, existe o tipo de varíavel const, que recebe um valor que é imutável - ou seja, não se alterará posteriormente.
Exemplo:
const BIRTHDAY = '30.07.1975'; //a data de aniversário é um valor constante, ou seja, que nunca muda, logo seu tipo de varíavel é const
