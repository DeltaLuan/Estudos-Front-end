Estilo em uma página HTML

Em HTML, é possível mudar a aparência dos elementos utilizando o atributo style.

Com este atributo, é possível mudar desde o tamanho do texto,

a fonte do texto

e até o alinhamento, entre muitas outras coisas.

Obs 1: As propriedades que o atributo style adiciona já é um tipo de CSS, porém o mais adequado é utilizar um arquivo .css separado do documento.

Obs 2: Nesse arquivo em específico foi utilizado uma forma "arcaica" de estilização, pois foi utilizado o style em cada uma das tags.
O mais adequado seria estilizar uma classe especifica para tags especificas, evitando assim de estender demais um código que pode ser diminuido.
